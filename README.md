# Premier League Strikers Data: 2025/2026 Season Analysis

## üéØ Project Overview

This repository serves as the central data hub for a statistical scouting and comparison project focusing on top strikers (center-forwards) expected to feature in the Premier League for the 2025/2026 season.

The project uses detailed, per-90-minute metrics from the preceding season to provide a multi-faceted view of player performance beyond simple goals and assists. The data allows for sophisticated analysis in areas such as:

* **Goal Efficiency:** Comparing Goals per 90 against Expected Goals (xG per 90) to measure finishing quality.
* **Off-Ball Contribution:** Analyzing defensive duels, interceptions, and pressuring success.
* **Playmaking:** Evaluating progressive passes, key passes, and expected assists (xA).
* **Presence:** Assessing touches in the box and aerial duel dominance.
